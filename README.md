# 🎓 Student Placement Prediction

## 📌 Project Overview

This project predicts whether a student will get placed based on academic performance and other features using Machine Learning.

The model is built using Logistic Regression and trained on a real student placement dataset.

---

## 🚀 Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Encoding
- Train-Test Split
- Logistic Regression Model
- Model Evaluation
- Accuracy Score
- Confusion Matrix

---

## 🛠 Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Dataset

Dataset: **Placement_Data_Full_Class.csv**

It contains student information such as:

- Gender
- Secondary Education Percentage
- Higher Secondary Percentage
- Degree Percentage
- Work Experience
- Specialisation
- MBA Percentage
- Placement Status

---

## 🤖 Machine Learning Workflow

1. Import Dataset
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Engineering
5. Train-Test Split
6. Model Training
7. Prediction
8. Model Evaluation

---

## 📊 Model Used

- Logistic Regression

---

## 📈 Result

The model predicts whether a student is likely to be placed based on academic and educational features.

### Clone the repository


```bash

```

### Run the notebook

Open **student_placement_prediction.ipynb** in Google Colab or Jupyter Notebook and run all cells.
